# BreakBuddy
O **BreakBuddy** é um aplicativo Android desenvolvido para ajudar usuários a gerenciar pausas durante o dia. Ele envia notificações lembrando o usuário de fazer pausas para atividades como tomar água, alongar-se ou descansar os olhos. O app no momento está em Work In Progress, ele foi criado usando **Kotlin** e **Jetpack Compose**, seguindo as melhores práticas de desenvolvimento Android.

---

## Futuras Funcionalidades

- **Adicionar Lembretes**: Crie lembretes personalizados para pausas ao longo do dia.
- **Notificações**: Receba notificações no horário programado.
- **Personalização**: Defina intervalos de tempo e tipos de pausas (água, alongamento, descanso, etc.).
- **Interface Moderna**: Design intuitivo e responsivo com Jetpack Compose.

---

## Tecnologias Que Serão Utilizadas

- **Kotlin**: Linguagem principal do projeto.
- **Jetpack Compose**: Framework moderno para construção de interfaces Android.
- **Android Notifications**: API para enviar notificações ao usuário.
- **Room Database**: Armazenamento local de lembretes e configurações (se necessário).

## Como Executar o Projeto

### Pré-requisitos
- Android Studio (versão mais recente).
- Dispositivo Android ou emulador configurado.

### Passos para Execução

1. Clone o repositório:
   ```bash
   git clone https://github.com/Boudenzin/BreakBuddy.git
   ```

2. Abra o projeto no Android Studio:
   - Selecione "Open an existing project" e navegue até a pasta do projeto.

3. Execute o app:
   - Conecte um dispositivo Android ou inicie um emulador.
   - Clique em "Run" (ícone de play) no Android Studio.
